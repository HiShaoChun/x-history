## 🎯 核心算法

### 热力图生成
```javascript
1. 生成最近 56 天日期数组
2. 遍历推文，统计每天浏览量
3. 找出最大值 maxCount
4. 归一化：level = (count / maxCount) * 4
5. 渲染 Grid 单元格
```

### 上瘾检测
```javascript
1. 按时间排序所有推文
2. 滑动窗口：15分钟
3. 计数窗口内推文数
4. if count >= 10: 标记为"上瘾时刻"
5. 跳过已统计的推文
```

### 作者统计
```javascript
1. 按 author 分组
2. 计算每个作者：
   - 浏览量 count
   - 点开率 opened/count
   - 平均停留 totalDwell/count
3. 排序取前 20 名
4. 映射到气泡图坐标
```

## 💡 设计哲学

### 数据密度
- 热力图：56 个数据点
- 雷达图：24 个数据点
- 气泡图：20 个数据点
- 上瘾分析：7 天 × 24 小时

### 交互原则
- **即时反馈**：鼠标悬停立刻显示详情
- **视觉层次**：颜色深浅表示重要程度
- **简洁明了**：一眼看出核心信息

### 性能优化
- 客户端计算（无网络请求）
- 限制数据展示范围
- 使用 Canvas 渲染（Chart.js）
