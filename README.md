# X History Recorder

一个 Chrome 浏览器扩展，用于记录你在 x.com（Twitter）上的浏览历史。

## 功能特点

- **自动记录**：无需手动操作，自动记录你看过的推文
- **智能判断**：
  - 推文在视口内停留 ≥ 1.5 秒才算"看过"
  - 点击进入详情页会标记为"已点开"
  - 页面失焦时暂停计时，避免挂机产生虚假数据
- **去重处理**：同一条推文只记录一次，重复浏览会累加停留时长
- **按时间查看**：按天和小时聚合展示，方便回溯
- **纯本地存储**：所有数据存储在本地 IndexedDB，不上传任何地方

## 记录的信息

每条推文记录包含：
- 作者
- 推文正文（摘要）
- 推文链接
- 首次看到时间 / 最后看到时间
- 累计停留时长
- 浏览次数
- 是否点开过详情页

## 安装方法

1. 下载或克隆此仓库
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目的文件夹

## 使用方法

1. 安装后，扩展会自动在 x.com 上工作
2. 正常浏览 X（Twitter），扩展会在后台自动记录
3. 点击扩展图标查看统计信息
4. 点击"查看完整历史"打开详细记录页面

## 数据管理

- 数据无限累积，不会自动清理
- 可在查看页面手动清除所有数据
- 所有数据仅存储在本地浏览器中

## 技术栈

- Chrome Extension Manifest V3
- Vanilla JavaScript
- IndexedDB

## 隐私说明

- 不调用任何 X 官方 API
- 不上传任何数据到服务器
- 纯被动观察页面 DOM 结构
- 所有数据仅存储在你的本地浏览器中

## 注意事项

1. 依赖 x.com 的 DOM 结构，如果 X 更新前端可能导致抓取失效
2. 仅支持 Chrome 浏览器（基于 Manifest V3）
3. 数据仅在当前浏览器中保存，不支持跨设备同步

## 开发说明

