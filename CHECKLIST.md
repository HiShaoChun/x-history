## 📋 使用前准备

1. **确认环境**
   - Chrome 浏览器（支持 Manifest V3）
   - 有 x.com 或 twitter.com 账号

2. **安装步骤**
   ```
   1. 打开 chrome://extensions/
   2. 开启"开发者模式"
   3. 点击"加载已解压的扩展程序"
   4. 选择本项目文件夹
   ```

3. **验证安装**
   - 扩展列表中显示 "X History Recorder"
   - 状态为"已启用"
   - 工具栏显示扩展图标
